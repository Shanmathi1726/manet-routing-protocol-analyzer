# MANET Routing Protocol Analyzer

An interactive MANET (Mobile Ad Hoc Network) Routing Protocol Analyzer built using HTML, CSS, and JavaScript to simulate and compare routing protocols including AODV, DSR, DSDV, and OLSR.

## Features
- Real-time MANET network simulation
- Interactive packet transmission visualization
- Comparison of reactive and proactive routing protocols
- Live performance metrics:
  - Packet Delivery Ratio (PDR)
  - End-to-End Delay
  - Throughput
  - Routing Overhead
- Dynamic event logs and analytics dashboard
- Adjustable simulation parameters:
  - Node count
  - Mobility
  - Traffic load

## Supported Protocols
- AODV (Ad hoc On-Demand Distance Vector)
- DSR (Dynamic Source Routing)
- DSDV (Destination-Sequenced Distance Vector)
- OLSR (Optimized Link State Routing)

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Canvas API

## Project Objective
The project aims to visualize and analyze the behavior and performance of different MANET routing protocols under varying network conditions through an interactive simulation environment.

## How to Run
1. Download or clone the repository
2. Open `index.html` in any modern web browser
3. Run the simulation and explore protocol performance
